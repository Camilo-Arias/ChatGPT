### Este proyecto busca conocer ChatGPT con Python

Para iniciar el proyecto necesitarás:

#### Instalar:
		pip install openai
		pip install "typer[all]"
		pip install rich
		

#### Generar un archivo config.py
Este archivo contendrá lo siguiente: 
		api_key = "Token generate openai.com"
